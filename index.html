📲 How to Install as App

A PWA (installable app) needs HTTPS. The free way is GitHub Pages — takes 3 minutes.

Step 1 — Create a free GitHub account
  • Go to github.com and sign up (free)
  • Verify your email
Step 2 — Create a repository
  • Click + → New repository
  • Name it: vaultkeeper
  • Set to Public, tick "Add README"
  • Click Create repository
Step 3 — Upload the file
  • Click "Add file" → Upload files
  • Upload VaultKeeper-Android.html
  • Rename it to index.html
  • Click Commit changes
Step 4 — Enable GitHub Pages
  • Go to Settings → Pages
  • Source: Deploy from branch → main → / (root)
  • Click Save
  • Wait 1-2 minutes for deployment
  • Your URL: yourusername.github.io/vaultkeeper
Step 5 — Install on Android
  • Open the URL in Chrome on your phone
  • Tap ⋮ menu → "Add to Home screen"
  • Tap Add — it installs like a real app
  • Your passwords stay encrypted in your browser — GitHub only hosts the app code, never your data
